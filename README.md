## 🎯 Project Overview

FinanceAI is a **production-grade financial intelligence platform** that analyzes any stock in the world using a combination of:

- 🧠 **LLaMA 4** (via Groq) for AI-powered investment analysis
- 📉 **LSTM Deep Learning** for 7-day price forecasting
- 🔍 **Z-Score Statistical Analysis** for anomaly detection
- 📰 **Real-time news** integration via Yahoo Finance
- 🗄️ **RAG Pipeline** with ChromaDB for knowledge retrieval
- 🔐 **Firebase Authentication** for secure user access
- ☁️ **Firestore** for storing user interactions and history

Supports stocks from **any market worldwide** — US, India (NSE/BSE), UK, Germany, Japan, and more.

## ✨ Features

### 🤖 AI-Powered Stock Analysis
- Full investment report generated by **LLaMA 4** (Meta's latest model)
- Sentiment analysis — Bullish / Bearish / Neutral
- Key strengths and risks identification
- Price target recommendation
- Powered by **real-time stock data + live news**

### 📈 LSTM Price Forecasting
- **7-day future price prediction** using deep learning
- 2-layer LSTM neural network with Dropout regularization
- Predicted vs Actual comparison table
- **Auto-refreshes every 60 seconds**
- ~2.45% Mean Absolute Percentage Error

### 🔍 Anomaly Detection
- **Z-Score statistical analysis** detects unusual price movements
- Identifies market crashes and spikes automatically
- Color-coded Crash 🔴 / Spike 🟢 indicators
- Auto-refreshes with live market data

## 🚀 Running Locally

### Prerequisites
- Python 3.10+
- Flutter 3.x
- Groq API Key (free at [console.groq.com](https://console.groq.com))
- Firebase project

### Backend Setup

```bash
# Clone the repository
git clone https://github.com/your_username/FinanceAI-Backend.git
cd FinanceAI-Backend

# Install dependencies
pip install -r requirements.txt

# Add your Groq API key in main.py
# Replace "your_groq_key_here" with your actual key

# Run the API
uvicorn main:app --reload --port 8000
```

API will be available at `http://localhost:8000`

### Frontend Setup

```bash
# Clone the Flutter repo
git clone https://github.com/your_username/FinanceAI-Flutter.git
cd finance_ai_flutter

# Install dependencies
flutter pub get

# Update API URL in lib/services/api_service.dart
# Change baseUrl to "http://localhost:8000"

# Run the app
flutter run -d chrome
```

## 🧠 ML Models

### LSTM Price Forecasting
- **Architecture:** LSTM(32) → Dropout(0.2) → Dense(16) → Dense(1)
- **Training data:** 1 year of daily closing prices
- **Sequence length:** 60 days
- **Train/Test split:** 80/20
- **Error rate:** ~2.45% MAPE
- **Forecast horizon:** 7 trading days

### Anomaly Detection
- **Method:** Z-Score Statistical Analysis
- **Threshold:** 2.0 standard deviations
- **Window:** 3 months of daily returns
- **Output:** Crash / Spike classification with severity score
